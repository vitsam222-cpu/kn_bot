# Telegram-бот: тестовый каркас (1 сообщение)

Это минимальный каркас Telegram-бота, который при `/start` отправляет **одно приветственное сообщение** с кнопками.

## 1) Последовательность шагов

1. Создать бота через [@BotFather](https://t.me/BotFather) и получить `BOT_TOKEN`.
2. Склонировать/подготовить проект и установить зависимости:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Скопировать `.env.example` в `.env` и заполнить токен.
4. Запустить бота:
   ```bash
   python bot.py
   ```
5. Открыть бота в Telegram и отправить `/start` — бот пришлёт одно тестовое сообщение.

## 2) Какие данные нужны от вас

Минимально:
- `BOT_TOKEN` (обязательно).

Опционально для следующего этапа:
- Точный текст приветствия.
- Список кнопок и их действия (ссылки/меню/консультация).
- Имя/username бота.
- Язык(и) интерфейса.

## 3) Что делает текущий каркас

- Обрабатывает только команду `/start`.
- Отправляет одно сообщение с тремя кнопками:
  - `📗 Шаблоны документов`
  - `📅 Курсы`
  - `🔷 Консультация`
- На нажатия пока не реагирует (только визуальный каркас).

## 4) Быстрый старт

```bash
cp .env.example .env
# вставьте токен в .env
python bot.py
```
